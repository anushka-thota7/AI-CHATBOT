# 🤖 AI Chatbot using Flask and API

## 📌 Project Overview

This project is a simple **AI Chatbot Web Application** built using **Python Flask, HTML, CSS, and JavaScript**.
The chatbot connects to an **AI API** to generate responses and allows users to chat through a clean web interface.

The application works similar to a basic chat system where users can send messages and receive AI-generated responses instantly.


## 🚀 Features

* AI-powered chatbot responses
* Flask backend integration
* Interactive chat interface
* Simple and responsive UI
* API-based AI response generation


## 🛠️ Technologies Used

* **Python**
* **Flask**
* **HTML**
* **CSS**
* **JavaScript**
* **AI API**

## 💬 How It Works

1. User types a message in the chat interface.
2. JavaScript sends the message to the Flask backend.
3. Flask sends the request to the AI API.
4. The API generates a response.
5. The response is returned and displayed in the chat interface.
